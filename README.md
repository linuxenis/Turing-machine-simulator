A simple turing machine simulator that reads text file input.
The text files contain some examples of input - 1 or 2 initial tapes and the definitions of the states.

what each sample input file does:

"1.txt" - converts the binary number on the left to hexadecimal on the right.

"2.txt" - adds two binary numbers and outputs the result on the right.

"3.txt" - binary "clock", adds 1 to the binary number continuously.

"4.txt" - a line of 1s that bounces in the space between the other two boundary 1s.

"5.txt" - reverses the string of 1s and 0s from tape 1 and puts it into tape 2.

"6.txt" - a bouncing bar on each tape.
